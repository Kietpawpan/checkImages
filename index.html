Machine Learning

โปรแกรมจำแนกวัตถุ พืช และสัตว์ในไฟล์รูปภาพ ด้วย machine learning โดยใช้ neural network (pre-trained MobileNet) จาก ml5.js library


เทคนิคการจำแนกภาพ

เป็นการจำแนกภาพ ด้วยเทคโนโลยี image recognition โดยใช้ ml5 library เวอร์ชันล่าสุด (1.0.1) ซึ่งอาศัย deep neural network ของ Google (Mobilenet network) มีความถูกต้องประมาณ 70% ทำงานโดยตรวจจับรูปร่างหน้าตาของวัตถุในภาพ แล้วจำแนกเป็นพืช สัตว์ หรือวัตถุอื่น ๆ พร้อมค่าความเชื่อมั่น และแปลผลเป็นภาษาไทย ด้วย Google Translate
